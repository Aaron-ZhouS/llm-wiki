# You need to know what you want

## 一句话总结

如果不知道自己真正想要什么，就很容易被平台、他人目标和短期刺激牵着走。

## NotebookLM 式知识信息图

```mermaid
flowchart TD
  A["知道自己想要什么"] --> B["过滤机会"]
  A --> C["选择行动"]
  A --> D["拒绝干扰"]
  A --> E["形成方向"]
  B --> B1["哪些机会适合我"]
  C --> C1["今天做什么"]
  D --> D1["不被比较带偏"]
  E --> E1["长期一致性"]
```

## 核心观点

1. 清晰目标是过滤器，没有目标时一切建议都显得有道理。
2. 想要什么不是空想，而是通过行动、反馈和复盘逐渐明确。
3. 对创作者来说，目标决定选题、产品、受众和生活方式。
