# 普通人改变命运：只有一个答案 — 复用

> 努力本质不是"忙"，而是做完以后——还能不能再用一次。

## 核心界定

普通人改变命运的终极答案：**复用**。
忙 ≠ 努力 ≠ 富。银行看你很穷，因为干完就着，没沉淀成资产。

## 系统机制

- **默认谬误**：以为忙就是努力，努力就能富。错了——忙完就着，没资产。
- **关键杠杆**：所有事都做可复用的。一次投入，多次产出，时间复利。
- **因果链**：干完就着 → 没资产 → 时间过去依然穷；干完能复用 → 一次投入多次产出 → 资产滚雪球。

## 资产型 vs 消耗型

| 资产型（加大投入） | 消耗型（砍掉/外包/找复用点） |
|-------------------|-----------------------------|
| 智能体自动回复评论区 | 胡子哥早期手回每条私信 |
| 模板化文案 / SOP / 知识卡片 | 单次咨询 / 一次性方案 |
| 投资思维能力复用 | 无 SOP 的重复劳动 |
| 短视频脚本沉淀为方法论 | "努力感"很强但没沉淀 |

## 边界条件

- **成立前提**：事情有可标准化成分，能抽象成流程或模板
- **断裂点**：纯一次性事件、完全无规律的私人定制——无法抽象成资产

## 最小动作：资产四问（每周日晚上）

1. **列清单** — 列出本周最忙的5件事
2. **判属性** — 逐个判断：可复用（资产）还是干完就着（消耗）
3. **调分配** — 资产型加大投入；消耗型砍掉/外包/抽象成模板
4. **月复审** — 每月/季度复盘：哪些可以不/少做？哪些加大投入？
