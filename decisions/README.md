# 决策百科（Decision Journal）

> 来源：视频《6个改变生活的 Hermes Agent 应用案例》❹ 场景（个人记忆 Wiki），改造为**一人公司决策日志**。

## 是什么

记录我**做过的所有关键决策** + **当时怎么想的** + **半年后回头看对不对**。

目的：当我再面对类似决策时，能调出自己过去的判断来参考（**不踩自己踩过的坑**）。

## 与对话日志的区别

| 维度 | 对话日志（Alex Finn 原版） | 决策百科（这版） |
|---|---|---|
| 记录对象 | 跟 AI 聊过什么 | 我**做过什么选择** |
| 价值 | 低（不查聊天） | 高（决策辅助） |
| 触发词 | "记得我们说过..." | "我上次是怎么决定的..." |
| 时间颗粒 | 每次会话 | 每次重大决策 |

## 什么时候记一条

满足**任一**条件就建一页：

- [ ] **方向选择**（赛道 / 主业 / 副业 / 退出某个事）
- [ ] **合作选择**（跟谁合作 / 拒绝谁的邀请）
- [ ] **资源投入**（花钱 / 投入时间超过 1 周 / 装修某个工具链）
- [ ] **对外承诺**（答应别人的事 / 公开说出去的观点）
- [ ] **拒绝选项**（主动放弃什么 — 这种最容易忘，反而最该记）

## 决策模板

见 [[decisions/_template]]。每条决策必须填 6 个字段：

1. **决策时点**（日期）
2. **当时考虑的选项**（≥ 2 个）
3. **最终选了哪个 + 为什么**
4. **预判风险**（当时觉得会出什么问题）
5. **半年/一年后复盘**（回填字段 — 半年后回来填）
6. **下次类似决策怎么参考**（这步是核心价值）

## 怎么用

**记录新决策**（3 选 1）：
1. 复制 `decisions/_template.md`，填 6 个字段，存为 `decisions/YYYY-MM-DD-简述.md`
2. 微信/飞书发我："记个决策：..."，我帮你按模板填好
3. 我检测到你说"我决定..."的时候主动问你"要不要建一条决策记录"

**查询旧决策**（2 选 1）：
1. 微信/飞书发我："我上次关于 X 是怎么决定的？" → 我查 wiki
2. 直接 Obsidian 搜 `[[decisions/`

## 半年后复盘的提醒

- 每天 evening-review 检查：有没有 6 个月前的决策到时间要回访？
- 触发词："半年后复盘" → 我跑这个流程
