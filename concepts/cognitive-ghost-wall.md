# 认知鬼打墙检测 (Cognitive Ghost Wall Detector)

> "学习的本质不是输入，是变异。"
> "不是你不努力，是你太擅长学习了。"
> — 姜胡说（胡子直播，原话 2024）

## 概念定义

胡子的原话：

> 一个人特别爱学，特别努力。你走了很远，累得呼哧呼哧喘，结果发现你绕了一圈又回到了原地。你学的所有知识都是在加固你原有的认知框架。你的系统会过滤掉不符合它自身的信息——它不阻止你学习，它还鼓励你学习，但你所有学习的最终结果，都是证明你原来的框架是对的。越学越封闭。

### 与其他概念的区分

| 概念 | 定义 | 与鬼打墙的区别 |
|------|------|---------------|
| 知识的诅咒 | 你知道了一个东西，就忘了别人不知道 | 静态的，一次性 |
| **知识鬼打墙** | 你有行动、很努力、在学，但原地转圈 | **动态的，更难发现** |
| 确认偏误 | 倾向于找支持自己观点的证据 | 偏认知层面 |
| **知识鬼打墙** | 你的整个系统在加固自己 | **偏系统层面，包含了行动** |

### 变异检测原则（核心心法）

做任何决策/学习/输入前，问自己：

> **"这个是基于新认知作出的选择，还是在加固旧认知？"**

如果答案是后者 → 停下来，切换输入源。

---

## 三合检测法

### 检测一：每日一问（30秒，每晚必做）

直接问自己：

> **"今天有没有发现我之前是错的？"**

| 结果 | 判定 | 下一步行动 |
|------|------|-----------|
| 有具体答案 | 今天有变异 ✅ | 继续保持，3句话记下来 |
| 想了30秒没有 | 今天在鬼打墙 ⚠️ | 第1天预警，N≥3激活检测二 |
| 连续3天没有 | 鬼打墙嫌疑 ❌ | **立即执行检测二和检测三** |

**为什么这个问题有效？**
发现"之前是错的" = 你的认知框架被刷新了。
没有 → 你所有的输入都在加固旧框架。

**容易犯的错：**
- ❌ 用"新学了东西"代替"发现之前是错的" — 学了不等于变了
- ❌ 有冲突感才叫变异："我之前一直以为X，现在发现X不成立/需要修正"
- ❌ 学一个顺应你旧观点的知识不是变异，是加固

---

### 检测二：强行失衡（30分钟，感觉停滞时用）

**核心方法：**

> 写下你现在深信不疑的观点，全网找相反的案例。

**操作步骤：**

1. **写下你深信不疑的观点**
   格式：`"我坚信[观点]，因为[理由]"`
   
   示例：
   - `"内容必须垂直才能火，因为算法推荐逻辑是标签匹配"`
   - `"只有学历高才能赚到钱"`
   - `"教人赚钱的都是骗子"`

2. **全网找相反案例（至少3个）**
   用 `web_search` 工具找：
   - 不垂直也火了的案例
   - 低学历高收入的案例
   - 免费教你赚钱的博主（胡子就是）

3. **对比分析**
   - 反例成立吗？还是特例？
   - 反例说明的规律跟你的观点有冲突吗？
   - 你的观点需要修正吗？修正成什么？

4. **记录变异点**
   如果发现自己之前的观点被修正了 → 这就是变异 ✓
   
   记录格式：
   ```
   我之前以为[X] → 现在发现其实[Y]，因为[反例Z]
   ```

**为什么有效？**
人类的认知框架天然会排斥矛盾信息。强行失衡就是"系统性给认知框架制造地震"。
如果你找不到反例 → **不是你的观点无懈可击，是你还没打开信息源**。

---

### 检测三：AI辅助扫描（有AI时用，每周一次）

让 AI 分析你的输入日志，评估同质化程度。

**详细 Prompt 模板：** 见 `templates/ai-scan-prompt.md`

**AI 输出格式：**
```
1. 同质化评估：
   - 新认知：[X]%
   - 加固旧认知：[Y]%
   - 无关：[Z]%

2. 回音壁检测：
   - 信息源单一程度：[1-10]
   - 回避的视角/领域：[列出]

3. 学习状态打分：
   - 变异分：[1-10]
   - 鬼打墙分：[1-10]
   - 综合建议：[具体行动]
```

## 跳出鬼打墙的3个方法

| 鬼打墙程度 | 判定标准 | 跳出方法 | 预计时间 |
|-----------|---------|---------|---------|
| **轻度** | 1-3天无变异 | 切换输入源——读一本跟你专业无关的好书，跟不同圈层的人聊天 | 1天 |
| **中度** | 1-2周无变异 | 执行完整的强行失衡检测 + 主动找一个你排斥的领域学习 | 1周 |
| **重度** | 1月+无变异 | **停止所有输入，转向输出**——写一篇文章/做一个作品/教一个人 | 1月 |

**根本原则（3条）：**

1. **输入不会自动变成变异。** 没有冲突感的输入都是旧认知加固。
2. **变异一定伴随着"之前是错的"的感觉。** 如果你学完感觉很舒服、很认同、很自洽——大概率在鬼打墙。
3. **输出的检测价值 > 输入的检测价值。** 真正能检验你有没有"变"的，是你做出来的东西。

## 检查点设计（用户确认节点）

| 检查点 | 触发条件 | 用户需要确认 |
|--------|---------|-------------|
| **CP1** | 检测一执行后（每日一问） | 是否进入检测二？（连续3天无变异自动激活） |
| **CP2** | 检测二执行后 | 反例是否成立？是否需要修正观点？ |
| **CP3** | 检测三执行后 | AI评分是否接受？是否进入跳出方案？ |
| **CP4** | 跳出方案选择 | 轻度/中度/重度？选定后必须执行至少一个动作 |
| **CP5** | 1周后回访 | 变异点是否真的内化？是否需要再次检测？ |

---

## 判断标准

| 状态 | 标准 | 下一步 |
|------|------|--------|
| 健康 ✅ | 每周至少2次发现"之前是错的" | 继续日常检测一即可 |
| 有嫌疑 ⚠️ | 连续3天无变异 / 同质化>70% | 执行检测二 |
| 确认鬼打墙 ❌ | 强行失衡找不到反例 / AI提示同质化>90% | 切换领域，强制输出 |

---

## 失败处理（Fallback）

| 失败场景 | Fallback |
|---------|---------|
| 找不到反例 | 不是你的观点无懈可击，是你还没打开信息源 → 换关键词 / 换平台 / 换语种搜索 |
| AI 评分不可信 | 手动执行检测一+检测二，AI 评分只作参考 |
| 跳出方案坚持不下去 | 降低难度：从不喜欢的领域换到无感的领域，最后换到轻松的输入 |
| 连续7天无变异 | 暂停所有 Skill 使用 3 天，纯体验生活 |

## 边界（什么时候不用）

- ✅ 适合：感觉学了很多但没变化时；每天都输入但说不出有什么改变时；AI分析你的输入同质化严重时；想主动保持认知健康时
- ❌ 不适合：
  - 日常学习输入过程中（先学，学完再检测）
  - 刚启动学习计划（还没积累足够的输入量，检测无意义）
  - 操作技能类学习（肌肉记忆不需要变异检测，直接用就好）
  - 纯娱乐性输入（不用检测，不是来学习的）

## 实操流程（完整版）

```
用户感觉停滞/想自检
       ↓
[CP1] 执行检测一（每日一问）
       ↓
   有变异？ ── 是 ──→ 记录变异点，继续学习
       ↓ 否
   连续3天无变异？
       ↓ 是
[CP2] 执行检测二（强行失衡）
       ↓
   找到反例？ ── 是 ──→ 修正观点，记录变异
       ↓ 否
[CP3] 执行检测三（AI辅助扫描）
       ↓
   AI确认同质化严重？ ── 是 ──→ 进入跳出方案
       ↓
[CP4] 判定程度：轻/中/重
       ↓
   执行对应跳出方法（至少1个动作）
       ↓
[CP5] 1周后回访验证
```
